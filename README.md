# Stackle

Low level soft stack and stack switching utilities.

## Status: pre-alpha

Not ready.

## Supported platforms:

* x86-64 unix

More to come:
* x86-64 windows
* x86 unix + windows
* aarch64 unix + windows
* arm unix + windows
* riscv unix
